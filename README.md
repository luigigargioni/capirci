# CAPIRCI

## Avvia server

`poetry run python manage.py runserver`  
OPPURE  
`poetry shell && python manage.py runserver`

---

## Installazione

`poetry install`  
OPPURE  
`poetry update`

Per aggiornare:
`poetry run poetryup`
